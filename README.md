This repository contains all my work completed during my **AI & Data Science Summer Training Program**.

The purpose of this repository is to document my learning journey through Python programming, Data Analysis, Data Visualization, Statistics, Machine Learning, and practical hands-on projects using Jupyter Notebooks.

Every notebook demonstrates concepts learned during the training along with practical implementations.

## 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/KhushiSharma006/AI-DS-Summer-Training.git
```

### Navigate into the project

```bash
cd AI-DS-Summer-Training
```

### Install dependencies

```bash
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```
